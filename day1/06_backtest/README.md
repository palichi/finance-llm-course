## 📊 백테스팅 파라미터 설명

```
초기자본:   백테스팅 시작 시 가진 돈 (예: 10,000,000원)
수수료율:   주문당 수수료 (KIS 모의투자 기준: 0.015%)
기간:      분석할 과거 기간 (예: 최근 1년)
전략:      어떤 규칙으로 매수/매도할지
```

---

## 🤖 자동매매 실행 옵션

```bash
python auto_trader.py \
  --ticker    005930   \   # 종목코드
  --strategy  ma_cross \   # 전략: ma_cross | rsi | bollinger
  --qty       1        \   # 1회 매매 수량 (주)
  --interval  60       \   # 체크 주기 (초)
  --max-loss  -5.0     \   # 손절 기준 (%)
  --take-profit 5.0        # 익절 기준 (%)
```

### 전략 종류
```
ma_cross   : 단기MA > 장기MA → 매수 / 단기MA < 장기MA → 매도
rsi        : RSI < 30 → 매수 / RSI > 70 → 매도
bollinger  : 하단밴드 터치 → 매수 / 상단밴드 터치 → 매도
```
