## 📐 기술지표 공식 (강의 참고)

### 이동평균 (Moving Average)
```python
MA5  = 종가.rolling(5).mean()   # 5일 이동평균
MA20 = 종가.rolling(20).mean()  # 20일 이동평균 (중기)
MA60 = 종가.rolling(60).mean()  # 60일 이동평균 (장기)
```

### RSI (상대강도지수, 0~100)
```python
delta = 종가.diff()
gain  = delta.clip(lower=0).rolling(14).mean()
loss  = (-delta.clip(upper=0)).rolling(14).mean()
RSI   = 100 - (100 / (1 + gain/loss))

# 해석
# RSI > 70 → 과매수 (하락 가능성)
# RSI < 30 → 과매도 (상승 가능성)
```

### MACD
```python
EMA12  = 종가.ewm(span=12).mean()
EMA26  = 종가.ewm(span=26).mean()
MACD   = EMA12 - EMA26
Signal = MACD.ewm(span=9).mean()
Hist   = MACD - Signal  # 히스토그램

# 해석
# MACD > Signal → 골든크로스 → 매수 신호
# MACD < Signal → 데드크로스 → 매도 신호
```

### 볼린저밴드
```python
MID   = 종가.rolling(20).mean()
STD   = 종가.rolling(20).std()
UPPER = MID + 2 * STD  # 상단밴드
LOWER = MID - 2 * STD  # 하단밴드

# 해석
# 종가 < 하단밴드 → 과매도 → 매수 고려
# 종가 > 상단밴드 → 과매수 → 매도 고려
```
